# Campus Parking Intelligence

Campus Parking Intelligence is a data and GIS project focused on helping students understand parking availability, restrictions, and parking difficulty around campus.

## Problem

Parking can be stressful for students, especially before class, during campus events, or when certain lots are restricted. Students often do not know which lots are realistic options, which ones are reserved, or which areas are likely to be difficult at certain times.

## Project goal

The goal is to build a data-driven parking helper that scores parking lots based on availability, restrictions, location, and time-based difficulty.

## Planned features

- Parking lot database
- Lot type classification
- Student, visitor, reserved, and event-restricted categories
- Parking difficulty score
- GIS map visualization
- Future recommendation system based on time and destination
